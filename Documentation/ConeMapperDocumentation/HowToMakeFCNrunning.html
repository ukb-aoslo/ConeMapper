How to make FCN running

Introduction

This instruction shows the way and problems with setup FCN on Windows OS.
You would be good to have an NVIDIA graphic card.
All steps listed here was tested with Matlab R2022b, Python 3.9.13, CUDA Toolkit v11.8 (only if you have NVIDIA)

Installation

First step is to install Python 3.6+. With the latest Python some of the libraries can be not compatible.
With "pip install" command in terminal install the next libraries:
matplotlib==3.6.2
numpy==1.23.5
scikit_image==0.19.3
scikit_learn==1.2.1
scipy==1.10.0
tqdm==4.64.1
tensorboard==2.11.2
For the pytorch library, use the next command:
pip install torch==2.0.1+cu118 torchvision==0.15.2+cu118 torchaudio==2.0.2 -f https://download.pytorch.org/whl/cu118/torch_stable.html
If you do not have NVIDIA:
pip install torch==2.0.1 torchvision==0.15.2 torchaudio==2.0.2 -f https://download.pytorch.org/whl/cu118/torch_stable.html
That is it. Now it should work.
