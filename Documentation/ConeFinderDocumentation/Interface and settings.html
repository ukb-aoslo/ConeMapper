Interface and settings

The main interface of the program divided on the sections.
Each section allows to manipulate different properties, or metrics.

File menu

File menu consists from the next options:
  • New recognition - starts the new cone recognition. More info in the Cone recogition section.
  • Open recognition - opens a .mat file with previously saved data.
  • Save recognition - saves the current recognition.
  • Settings - calls the general settings window.
  • Exit - closes the app.
Data usually saves in the same file that was opened, or in mat file with the same name and given image, if it is a new recognition.

Settings

In the settings menu you can setup startup folder for files navigation and turn on/off the autosave.
These settings will be written to "settings.mat" file.
Autosave option (if turn on) saves the data each 5 minutes of time to the file "<your_current_filename>_autosave.mat" to the same location as your .mat file or image file.

Import/Export menu

Import/Export menu consists from the next options:
  • Import cone locations
  • Export cone locations
More about it, read in Import/Export section.

View menu

In view menu you can select only a color map. Color map influences the Voronoi Diagram colors and the colors for density map.
Available color maps:
  • Bone
  • Cividis
  • CoolWarm
  • Inferno
  • Magma
  • Plasma
  • Parula
  • Viridis

Metrics menu

Metrics menu consists from the next options:
  • Show stats window - shows the window with statistical data about the current image.
  • EuclidianNCones - calculates the density map based on Euclidian distance to nearby cones of given amount.
  • NNDmean - calculates the density map based on average distance to nearest neighbor cones.
  • YellotsRings - calculates the density map based on frequency analysis of the image.
  • Density Lines Settings - calls settings window, which allows to changes properties of density contour lines.
  • Units Settings - calls settings window, where user can choose the units for density data or scale bar.
More about it, read in Metrics section.

Help menu

Help menu consists from the next options:
  • Show documentation - shows this documentation
  • About... - shows short info about the app.
